Задача

Реализуйте статический метод для решения задачи ЛР3#5789(открыть в новой вкладке)

  1. В папке для исходного кода создайте класс Point.
  2. В классе Point реализуйте публичный статический метод distanceBetween.
  3. Он принимает в качестве аргумента 4 действительных числа и возвращает действительное число.
  4. В теле метода верните результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы метода с помощью юнит-теста

  1. В папке для юнит-тестов создайте класс PointSuite.
  2. В классе PointSuite создайте юнит-тест distanceBetweenValuesTest.
  3. Для каждой строки тестовых данных из задачи напишите в теле юнит-теста фрагмент кода:
    1. Вызовите статический метод distanceBetween класса Point.
    2. Передайте ему в качестве аргументов значения Xa, Ya, Xb и Yb.
    3. Добавьте выражение для проверки результата вызова метода distanceBetween.
  4. После запуска тестового метода, он должен быть отмечен зеленым знаком в списке методов.
